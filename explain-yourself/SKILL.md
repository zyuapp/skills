---
name: explain-yourself
description: >
  Explain recent code work by reading the branch diff against the base branch,
  then summarizing what changed and why in a human update style. Use this skill
  when the user asks what was done, why an approach was taken, or wants an
  explanation of the current branch without a line-by-line code recap.
---

Read the branch diff against the base branch. Explain what you did and why, as if you are giving an update to the person who requested the work.

Focus on the reasoning behind the approach, not a line-by-line recap of the code.
